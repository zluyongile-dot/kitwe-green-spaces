Kitwe Green Space Mapping System - ICT431 Capstone Project

Project Overview

A comprehensive GIS-based web application for mapping, analyzing, and managing urban green spaces in Kitwe, Zambia. This system demonstrates advanced web development, spatial analysis, and data visualization techniques.

Technical Architecture

Technology Stack
Python Flask PostgreSQL PostGIS JavaScript ES6+ Leaflet.js Bootstrap 5 Chart.js HTML5/CSS3 RESTful API GeoJSON
System Architecture
Frontend (Client-Side) ├── HTML5 Semantic Structure ├── CSS3 with Flexbox/Grid ├── JavaScript (ES6+) ├── Leaflet.js (Mapping) ├── Chart.js (Analytics) └── Bootstrap 5 (UI Framework) Backend (Server-Side) ├── Python Flask Framework ├── PostgreSQL Database ├── PostGIS Spatial Extension ├── RESTful API Endpoints └── CORS Configuration Data Layer ├── Spatial Database (PostGIS) ├── GeoJSON Data Format ├── Real-time API Integration └── Advanced Analytics Engine
Database Schema
CREATE TABLE green_spaces ( id SERIAL PRIMARY KEY, name VARCHAR(255) NOT NULL, type VARCHAR(100), area_sq_m FLOAT, ward VARCHAR(100), geom GEOMETRY(Point, 4326) ); CREATE TABLE public_feedback ( id SERIAL PRIMARY KEY, green_space_id INTEGER REFERENCES green_spaces(id), user_name VARCHAR(100), user_email VARCHAR(100), issue_type VARCHAR(50), description TEXT, status VARCHAR(20) DEFAULT 'pending', created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP );

Environmental Impact Methodology

CO₂ Absorption
CO₂ = Area × 0.0002 kg/m²/year Based on: - Urban tree coverage - Grass carbon sequestration - Soil organic carbon
Rainwater Management
Capacity = Area × 0.001 L/m² Factors: - Soil infiltration rate - Vegetation water retention - Surface permeability
Urban Cooling
Cooling = Area × 0.00001 °C/m² Based on: - Evapotranspiration - Shade provision - Albedo effects
